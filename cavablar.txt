Düzgün oxunaqlı CSS kod yazmaq qaydaları:
1) CSS Reset kimi toollardan istifadə etmək

2) Düzgün və başadüşülən class yaxud id istifadə etmək

3) Kod yazmazdan öncə plan qurmaq

4) Eyni vertical və yaxud səliqəli sırada yazmaq

5) Bracketləri {} düzgün yerdə açıb və bağlamaq

6) HTML kod tam hazır olduqdan sonra CSS-də style verməyə başlamaq

7) Lazım olmadıqca şəkil istifadəsini azaltmalı əvəzində fontawesome, shape və digər toollardan istifadə etmək

8) CSS style yazdıqdan sonra ikinci,üçüncü dəfə üzərindən keçərək yazdığımız kodu optimizasiya etmək

9) Təkrar classlardan qurtulmaq, mümkün qədər az koddan istifadə etmək(təkrarlardan qaçmaq)

10) İşə könül vermək. Kodu sevgi ilə yazmaq.
